## Summary

**Maintenance patch:** refreshes CI GitHub Actions pins, Sovereign UI frontend npm dependencies, and Dependabot grouping — no operator-facing behavior changes beyond staying current on toolchain security fixes.

**Why upgrade:** keeps CI on supported Actions (`checkout@v7`, `dependency-review-action@v5`, `setup-uv@v8.2.0`) and frontend build deps on latest patch/minor releases. Dependabot now opens grouped weekly PRs instead of flooding the repo.

Operators should upgrade with `uvx matryca-plumber` ≥ **1.10.4**. Point LLM hosts at **`llms.txt` v1.10.4**.

## Changed

- **CI / GitHub Actions** — toolchain refresh across all workflows (metrics-saver aligned from v4).
- **Sovereign UI frontend** — npm dependency bumps; build/lint/test unchanged.
- **Dependabot** — grouped weekly updates (supersedes PRs #75–#82).
