# GitHub Release — v1.10.0 (copy-paste)

**Title:** v1.10.0 - Catalog Integrity & OSS Maturity

**Tag:** `v1.10.0`

---

## Summary

**Data integrity + maintainer maturity release:** closes the v1.9.10 **Concurrency & Data Integrity** milestone ([#35](https://github.com/MarcoPorcellato/matryca-plumber/issues/35)–[#37](https://github.com/MarcoPorcellato/matryca-plumber/issues/37), [#41](https://github.com/MarcoPorcellato/matryca-plumber/issues/41)) with flock-protected master catalog I/O, merge-on-save, atomic link registry writes, and harvest catalog/page parity on OCC abort. Adds GitHub Community Profile completeness (PR template, CodeQL, frontend ESLint in CI, dependency-review, release verify gate) and clears open Dependabot advisories.

**Why upgrade:** concurrent daemon/harvest writers no longer tear or clobber JSON sidecars; operators and contributors get a hardened supply chain and predictable CI.

Operators should upgrade with `uvx matryca-plumber` ≥ **1.10.0**. Point LLM hosts at **`llms.txt` v1.10.0**.

## Highlights

| Area | Change |
|------|--------|
| Master catalog (#35, #36) | Load/save under `cross_process_json_flock`; merge-on-save by `last_mtime` |
| Link registry (#41) | `atomic_write_bytes` persistence — no truncated JSON on crash |
| Bootstrap harvest (#37) | Skips catalog upsert when semantic index append OCC-aborts |
| OSS / CI | PR template, CODEOWNERS, CodeQL, frontend ESLint+Vitest, release verify job |
| Security | Transitive bumps for `starlette`, `aiohttp`, `cryptography`, `python-multipart`, frontend toolchain |

## Fixed

- **Master catalog load flock (#35)** — Torn reads eliminated on `master_catalog.json` and `.bak` restore.
- **Master catalog merge-on-save (#36)** — Last-writer-wins clobbering fixed under concurrent harvest/daemon writers.
- **Harvest catalog drift guard (#37)** — Catalog rows no longer claim summaries never written after OCC abort.
- **Link registry atomic save (#41)** — Registry writes match daemon state / backlink index durability.

## Changed

- **OSS / GitHub hygiene** — Community Profile complete; version-consistency guard; PyPI metadata enriched.
- **Sovereign UI** — React hooks refactored for ESLint react-hooks v7 compliance.

## Security

- **Dependency advisories** — Python and frontend toolchain bumps to clear open Dependabot alerts.
