## Summary

**Maintenance release:** prepares the repository for external contributors and lands scoped v1.9.x tech-debt fixes without a parser semver bump. Phase 2 now treats daily journals as a first-class flat cluster (fewer wasted cluster-focus tokens), entity consolidation skips nonsensical person-vs-date LLM turns, and the suite stands at **710+ passing tests**.

**Why upgrade:** journal-heavy vaults see leaner Phase 2 prompts; OSS contributors get [`good_first_issues_blueprints.md`](../../good_first_issues_blueprints.md) with six ready-to-paste issue comments; duplicate HTTP redirect handlers and alias-registry private mutations are centralized for easier review before v2.0 Shadow DB.

Operators should upgrade with `uvx matryca-plumber` ≥ **1.9.14**. Point LLM hosts at **`llms.txt` v1.9.14** (§2.4 · journal-aware clustering).
