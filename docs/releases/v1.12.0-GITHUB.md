## Summary

**Minor architecture release (plan v3):** Tier-1 daemon prompts follow Robert C. Martin–style boundaries — domain `*/prompts.py` builders, constructor injection on `InstructorLLMClient`, and **L0 hard rejection** before semantic index disk writes. Runtime cognitive law (`SYSTEM_PROMPT.md`) is assembled from OpenSpec fragments with CI `build-hash` guards.

**Maintainer contracts:** [`AGENTS.md`](../../AGENTS.md) three-audience router, `make agents-check`, `make build-system-prompt` / `make check-system-prompt`, Cursor rule `11-prompt-maintainer`.

Operators should upgrade with `uvx matryca-plumber` ≥ **1.12.0**. Point LLM hosts at **`llms.txt` v1.12.0**.
