## Summary

**Developer-experience patch:** tightens the local test gate (`make test-fast` ~5s) by excluding `@pytest.mark.integration` subprocess/bootstrap tests into `make test-integration`, and clears strict mypy on the semantic cache router TTL test.

**Why upgrade:** no runtime behavior change for operators on v1.10.0; contributors get a faster, CI-aligned local loop. If you never shipped v1.10.0 to PyPI, this tag includes the full catalog/registry integrity line from `[1.10.0]`.

Operators should upgrade with `uvx matryca-plumber` ≥ **1.10.2**. Point LLM hosts at **`llms.txt` v1.10.2**.

## Highlights

| Area | Change |
|------|--------|
| `make test-fast` | Skips integration-marked tests; no real sleeps in TTL/I/O-retry unit tests (~4–5s gate) |
| `make test-integration` | New target for subprocess/bootstrap integration slice |
| CI / mypy | Semantic cache router TTL test patches `time.time` via module path |

## Fixed

- **CI mypy** — `test_semantic_cache_router` no longer accesses `router.time` (attr-defined clean).

## Changed

- **`make test-fast`** — `@pytest.mark.integration` exclusion; adds `make test-integration` companion target.
